Raison d’être:
Ce dossier contient le fichier public utilisé par l’interface de l’application. Il regroupe les éléments comme les styles et autres ressources nécessaires à l’affichage dans le navigateur.

Contenu:
Le fichier principal est style.css. Il définit l’apparence de la calculatrice. On y retrouve la mise en page, les couleurs, les polices et le style des boutons afin d’avoir une interface claire et cohérente.

Dépendances et hypothèses:
Le fichier qu’il contient sont utilisés dans la page HTML et doit être appelé à l’aide de la fonction url_for de Jinja2 pour être chargés correctement.
